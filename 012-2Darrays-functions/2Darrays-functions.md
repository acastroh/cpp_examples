# description
Ask for 9 float values to be stored in a 3x3 float bidimensional array, display it.
Then, call a function that receives the matrix as an input parameter and display it transposed.
Example:

| 8 | 9.2 | 10 | 
|---|---|---|
|5.1 | 4.6 | 7.6 |
|2.2 | 5 | 3 |

Transposed:
| 8 | 5.1 | 2.2 |
|---|---|---|
| 9.2 | 4.6 | 5 |
| 10 | 7.6 | 3 |
